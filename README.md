# Math Table UI

A colorful React-based application for children to practice and explore multiplication tables interactively using a FastAPI backend API.

---

## Features

- Interactive multiplication table generator
- Child-friendly colorful UI
- Fast and responsive design
- Real-time table generation
- Beginner-friendly educational project

---

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- FastAPI Backend API
- Render Deployment
